# CS360 A4 Report Layout V8

## Included changes

- The Special CS Client brand page now displays `Client: <client name>`.
- Removed `Special CS Client` wording from the generated report pages.
- View Report and Print / Save PDF now use the same organized A4 landscape layout.
- Brand and location details are automatically divided into separate A4 pages instead of extending outside the page.
- Completion charts, legends, KPI values, percentages, and table values remain inside their cards or cells.
- Long client, brand, group, and location names wrap safely within their widgets.
- Brand and location tables use controlled column widths for A4 landscape.
- Updated the `client-success.js` cache version in `index.html`.

## Installation

1. Replace `client-success.js` in the project root.
2. Replace `index.html` in the project root.
3. Redeploy the platform.
4. Press `Ctrl + Shift + R` to force the updated JavaScript to load.

No Supabase SQL migration is required for this update.
